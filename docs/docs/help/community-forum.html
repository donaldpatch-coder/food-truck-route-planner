TruckNet Community

Use TruckNet Community for questions, owner-to-owner help, and urgent needs at events.

What To Post

  • Use Quick Need for short requests like extra mustard, cups, propane, ice, or staff coverage.
  • Use Locations & Events for event feedback, parking tips, and contact notes.
  • Use Vendor Reviews for suppliers, repairs, or recommendations.
  • Use Equipment Help when owners need troubleshooting or repair advice.

Check-In Communication

When an owner checks into a location, they can choose to be visible to other owners. Visible check-ins show the business name, contact note, time, and location so owners can coordinate if needed.

The forum is stored in the browser for testing. A live version should move posts and visible check-ins into the shared database with moderation.
